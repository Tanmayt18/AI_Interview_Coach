# AI Interview Coach 🤖

AI powered interview preparation system with:

• AI generated interview questions  
• Coding interview mode  
• Resume-based questions  
• Automatic interview timer  
• AI evaluation and feedback  

## Tech Stack

- Python
- Streamlit
- Ollama (Llama3)
- NLP
- Speech Recognition

## Run Project

pip install -r requirements.txt  
streamlit run app.py
